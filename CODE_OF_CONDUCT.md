# Code of Conduct

## Our Pledge

We want this project to be an open, respectful, and constructive place for contributors of all
backgrounds and experience levels.

## Our Standards

Examples of behavior that help build a positive environment:

- being respectful and patient
- giving actionable technical feedback
- assuming good intent while still being clear
- welcoming questions from new contributors
- focusing discussion on the code, docs, and user impact

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- sexualized language or unwelcome attention
- publishing private information without permission
- hostile or intentionally dismissive review behavior
- spam, trolling, or bad-faith disruption

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and
other contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainer through GitHub.
Include links, screenshots, or other context when possible.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and
other repository-linked communication.
