# houjin-bangou-api-mcp

Model Context Protocol (MCP) server for the Japan National Tax Agency Corporate Number API.

## Status

This repository is being set up as a public starting point for an MCP server that wraps the
official Corporate Number Publication Site Web-API provided by the National Tax Agency of Japan.

## Scope

- Search corporate records by corporation number
- Search by company or organization name
- Fetch update history by date range
- Expose the API through MCP tools for LLM clients

## Notes

- The official API requires an application ID.
- The API documentation is available from the National Tax Agency:
  https://www.houjin-bangou.nta.go.jp/webapi/index.html
